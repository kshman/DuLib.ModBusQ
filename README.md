## 모드버스 클래스 라이브러리

### 소개
* 모드버스 프로토콜을 지원하는 클래스 라이브러리입니다.
* RTU, TCP 모드버스 통신을 지원합니다.
* 서버와 클라이언트 기능을 제공합니다.

### 주의
* 완전한 비동기 통신은 현재 지원하지 않습니다.
* 일부 기능은 완전하게 테스트되지 않았습니다.

## 클라이언트 사용법
```csharp
// TCP 모드버스 클라이언트를 생성하여 디바이스에 연결합니다.
using var c = new ModBusClientTcp("10.0.0.1", 5000);
c.Open();
// 연결이 성공적으로 이루어졌는지 확인합니다.
if (c.IsConnected)
{
	// 디바이스에서 10개의 레지스터를 읽어옵니다.
	var data = c.ReadInputRegisters(0, 10);
	// 읽어온 데이터를 처리합니다.
}
```
